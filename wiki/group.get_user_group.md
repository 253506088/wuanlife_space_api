# group.get_user_group

获取用户加入星球的接口

## 接口调用请求说明

接口URL：http://dev.wuanlife.com:800/group/get_user_group

请求方式：GET

参数说明：

|参数名字       | 类型 | 是否必须  |  默认值  |  范围   |                说明|
|:--|:--|:--|:--|:--|:--|
|user_id|整型|必须|-|-| 用户id|
|pn|整型|可选|1|-|当前页面|

## 返回说明

|返回字段                 |类型    |    说明|
|:--|:--|:--|
|groups.g_name   |           string     | 星球名称|
|groups.group_id          |      int  |       星球id|
|groups.g_image   |        string    |  星球图片|
|groups.g_introduction  |  string   |   星球介绍|
|groups.num          |      int      |   成员数量|
|page_count        |        int   |      总页数|
|current_page    |          int   |      当前页|
|num|整型|星球数量|
|user_name|字符型|用户昵称|
|msg|字符型|提示信息|
