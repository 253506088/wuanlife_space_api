1.API名称、描述
Post.UnStickyPost
取消置顶帖子接口

2.接口URL：
http://dev.wuanlife.com:800/?service=Post.UnStickyPost

3.请求方式：
POST

4.参数说明：
参数名字    类型   是否必须    默认值    范围        说明
user_id    字符串   必须               最小：1     用户id
post_id    字符串   必须               最小：1     帖子id

5.返回值说明：
返回字段        类型   说明
code            整型   操作码，1表示取消置顶帖子成功，0表示取消置顶帖子失败
re             字符串  提示信息

6.示例：
登录账号
http://dev.wuanlife.com:800/?service=Post.UnStickyPost&user_id=13&post_id=1
JSON:
{
    "ret": 200,
    "data": {
        "code": 1,
        "re": "操作成功"
    },
    "msg": ""
}
